GNUMP3d - The GNU Streaming MP3 / Media Server

GNUMP3d is a streaming server for MP3s, OGG vorbis files, movies and other media formats.

It is designed to be:

  • Small, stable, portable, self-contained, and secure.
  • Simple to install, configure, and use.
  • Portable across different varieties of Unix, the GNU Operating System, and Microsoft Windows platforms.

GNUMP3d v2.9final Released

The current stable release of GNUMP3d is v2.9final, which was released upon the 22nd of April 2007.

This new release features only minor changes, and is intended to be the final release of the 2.x branch of the project. In the future there might be a 3.x release branch using modern Perl or Ruby as the implementation language - however that is not yet decided.

In the past five years I've had an awful lot of fun working on this project, learned an awful lot of things, and feel the the world has moved on from my little project. It might be time to lay it to rest for good rather than start again with a real SQLite database backend, etc.

Please find the archives on the download page.

Contribute

In the past I welcomed the contribution of code, documentation, or theme designs - not to mention items from my Amazon wishlist. However with the current mothballing of the 2.x release I think it is no longer appropriate to accept such contributions - since that would mean making additional releases which is contrary to the idea of a "final" release.

I owe a large part of the success of this project to the many suggestions, and patches from previous contributors, their feedback, updates, and support is appreciated very much.
