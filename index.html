GNUMP3d - The GNU Streaming MP3 / Media Server

GNUMP3d is a streaming server for MP3's, OGG vorbis files, movies and other media. It is designed to be:

  • Small, stable, portable, self-contained, and secure.
  • Simple to install, configure, and use.
  • Portable across different varieties of Unix and Microsoft Windows platforms.

GNUMP3d v2.7 Released

The current stable release of GNUMP3d is v2.7, which was released upon March 25th 2004.

The current release features many small cleanups and changes - a complete list may be found in the ChangeLog. Prominent changes include:

  • Full support for internationlised interfaces.
  • The ability to browse your archive by tag information
  • Multiple theme cleanups.
  • Updated networking code.
  • Much faster index generation.
