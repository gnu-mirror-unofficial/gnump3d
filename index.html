GNUMP3d - The GNU Streaming MP3 / Media Server

 GNUMP3d is a streaming server for MP3's, OGG vorbis files, and other media. It is designed to be:

  • Small, stable, portable, self-contained, and secure.
  • Simple to install, configure, and use.
  • Portable across different varieties of Unix and Microsoft Windows platforms.

GNUMP3d v2.6 Available

The current release of GNUMP3d v2.6 is available for download again - after being in a state of hiatus due to the recent Savannah compromise.

Active development will be restarting now that the CVS repository is usable once more.

My apologies for any inconvience caused by my inability to update the code or these webpages.

All pending patches will be committed over the weekend at the latest - and questions may be posted to the mailing lists.

 The current release features many small cleanups and changes - a complete list may be found in the ChangeLog. Prominent changes include:

  • Full support for internationlised interfaces.
  • Several new plugins.
  • A new theme.
  • A "random directory" function
  • Improved diagnostic messages
