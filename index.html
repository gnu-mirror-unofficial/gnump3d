GNUMP3d - GNU MP3 Daemon

 GNUMP3d is a portable, stable, and simple MP3/OGG Vorbis streaming server, complete with an attractive web-based interface.

 Currently the project is hosted upon SourceForge, but it has recently been dubbed a GNU Program - and is in the process of migration to this site.

 Until the migration is complete you may look at:

  • The CVS Repository for the code.
  • The CVS Repository for these web pages.
  • The GNUMP3d Webpages - Held on SourceForge
