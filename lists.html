GNUMP3d Mailing Lists

There are several mailing lists which have been created for discussion related to gnump3d - below you can find descriptions of each of them.

gnump3d-users

This is a list for discussions by users of the software, its a handy place to make suggestions, share templates, and offer/receive assistance from users.

The list is archived here, you can subscribe/unsubscribe here

gnump3d-devel

This is a list for development discussions of the software.

The list is archived here, you can subscribe/unsubscribe here

gnump3d-commits

This is a list for CVS commit notifications for this software. Each commit to the programs CVS Repository will trigger a mail to this list.

The list is archived here, you can subscribe/unsubscribe here
